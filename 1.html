HTML

XHTML is(이란무엇인가) a variant of HTML that uses the syntax of XML, the Extensible Markup Language

.

XHTML has all the same elements (for paragraphs, etc.) as the HTML variant, but the syntax is slightly different.

Because XHTML is an XML application, you can use other XML tools with it (such as XSLT, a language for transforming XML content).
